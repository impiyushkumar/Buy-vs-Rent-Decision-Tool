## Problem Statement
Buying a home vs renting is an emotional decision. This tool helps users make a
data-driven choice by comparing long-term net worth outcomes.

## How It Works
The tool compares two scenarios over 20 years:
- Buying a home using a loan (EMI, down payment, property appreciation)
- Renting a home and investing the savings via SIP

## Financial Logic Used
- EMI calculation for home loan
- SIP future value using compounding
- Rent inflation modeling
- Property appreciation modeling
- Net worth calculation considering total cash outflow
- Break-even year identification

## Tech Stack
- Python
- Pandas
- Streamlit
- Plotly

## Features
- EMI vs Rent comparison
- 20-year net worth visualization
- Break-even year detection
- Clean summary metrics
- Interactive UI
